# 業務補助ツール PWA版

JAN管理、農薬検索、チェーンソー部品検索、電動工具替刃検索、ホース長さ計算をまとめた静的Webアプリです。

## 現在の版

`1.0.4`

## 1.0.4の変更

- 農薬検索に「複数検索」トグルを追加
- 選択したすべての作物に共通する農薬を、登録上の農薬名が完全一致する条件で抽出
- 共通農薬カード内に作物ごとの対象、希釈・使用量、散布液量、使用時期、回数、方法を表示
- 結果は30製品ずつ追加表示し、多数の共通農薬がある場合の描画負荷を抑制

## 1.0.3の変更

- 商品情報の「JANコード：数字」という文字表示を廃止
- 「バーコードを表示」ボタンを廃止
- バーコードとその下の数字を直接表示
- 画面の上下500px以内へ近づいたバーコードだけSVGを生成
- 同じJANのSVGはメモリ内キャッシュから再利用
- バーコードをタップまたはEnterキーでJANをコピー
- 農薬検索で多数の結果が並んでも、画面外のバーコードは未生成のまま維持

## 動作条件

カメラ、Service Worker、PWAインストールは、原則としてHTTPSまたはlocalhostで動作します。端末内のHTMLファイルを直接開く方法ではPWA機能を利用できません。

## 構成

- `index.html`: ツール一覧
- `manifest.webmanifest`: PWA設定
- `service-worker.js`: オフラインキャッシュと更新制御
- `version.json`: 現在の版情報
- `shared/js/ean13.js`: JAN検証、バーコード生成、遅延表示、コピー処理
- `tools/pesticide-search/js/multi-crop.js`: 複数作物の共通農薬抽出
- `tools/`: 各ツール
- `scripts/`: 検査スクリプト

## 保守時の注意

- 公開ファイルを変更した場合は、`version.json`と`service-worker.js`の版番号を同時に更新します。
- 農薬データは約10万件あるため、`data.js`全体の不要な自動整形を避けます。
- 商品情報にJANがない場合はバーコードを生成しません。
