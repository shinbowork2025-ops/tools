# 業務補助ツール PWA版

JAN管理、農薬検索、チェーンソー部品検索、電動工具替刃検索、ホース長さ計算、木材カット図をまとめた静的Webアプリです。

## 現在の版

`1.1.3-prototype`

## 木材カット図（試作版）

この機能は操作方法と計算条件を確認するための試作版です。ツール一覧と機能画面の両方に「試作版」と表示します。

- 長方形の材料を、縦または横の端から端までの直線で分割
- 数値入力とカット線のドラッグ調整
- 2本指の拡大・縮小と余白の移動
- 一つの図面に複数材料を登録
- カット順、切断後の部材寸法、寸法線を表示
- 元に戻す、やり直す、端末内への保存、再編集、印刷
- 材料マスタを `tools/wood-cut-planner/js/materials.js` に分離
- 専用カメラでJANを読み取り、登録材料と寸法を自動選択
- JANと寸法を確認できたコメリ取扱木材51商品を収録
- 杉材、杉板、桧材、胴縁、赤松材、ホワイトウッド間柱を収録
- 檜工作角棒5商品、一般丸棒3商品、米ヒバ丸棒4商品を収録
- 未登録JANは商品名・寸法とともに端末内へ記憶

現時点では部材の回転、斜めカット、切り欠き、自動配置、PDFファイルの直接生成には対応していません。ブラウザの印刷画面から、端末側の機能でPDFとして保存することは可能です。

丸棒はJAN照合と長さカットに利用できるよう、`width`と`thickness`へ直径、`height`へ長さを登録します。図面上の断面表示は円ではなく長方形による近似です。

木材商品は店舗・地域によって取扱いが異なるため、加工前に現物のJANと寸法表示を確認してください。公式商品ページでJAN・寸法を確認できないサイズ違いは、推測でマスタへ追加しません。

## 1.0.5の変更

- インストールしたPWAを縦向き表示へ固定
- JANスキャンボタンを画面下部中央へ固定
- JAN保存リストを最大6個のタブで管理
- タブ名は連番で作成し、選択中の名前を再タップして編集
- 旧単一リストを「リスト1」へ自動移行
- 各JANのバーコード横へメモ欄を追加
- 読み取り確定後にカメラを自動停止して一覧へ戻る

## 1.0.4の変更

- 農薬検索に複数作物モードを追加
- 選択した全作物に共通する農薬名を抽出
- 同じ農薬カード内へ作物別の適用条件を表示

## 1.0.3の変更

- 商品バーコードを直接表示
- 画面付近のバーコードだけ遅延生成
- バーコードのタップでJANをコピー

## 動作条件

カメラ、Service Worker、PWAインストールは、原則としてHTTPSまたはlocalhostで動作します。縦向き固定はインストールしたPWAで有効になり、通常のブラウザタブでは端末設定が優先される場合があります。

## 構成

- `manifest.webmanifest`: PWA設定と縦向き指定
- `service-worker.js`: オフラインキャッシュと更新制御
- `shared/js/ean13.js`: JANバーコードの遅延表示
- `tools/jan-scanner/js/list-store.js`: JAN複数リストの保存と移行
- `tools/pesticide-search/js/multi-crop.js`: 複数作物の共通農薬抽出
- `tools/wood-cut-planner/`: 木材カット図の試作版
- `tools/wood-cut-planner/js/jan-scanner.js`: 木材専用JANカメラ読取
- `tools/wood-cut-planner/js/materials.js`: JAN・寸法・確認元を含む木材商品マスタ
- `scripts/`: 検査スクリプト

## 保守時の注意

- 公開ファイルを変更した場合は、`version.json`と`service-worker.js`の版番号を同時に更新してください。
- 木材商品データは `tools/wood-cut-planner/js/materials.js` を差し替えて更新します。
- 商品マスタのJAN・寸法は現物や公式商品ページと照合し、確認日と出典を残してください。
