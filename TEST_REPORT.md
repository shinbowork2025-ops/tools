# 検査記録

実施日: 2026-06-26

## JANスキャンメモ（1.0.5）

- 旧単一リストを「リスト1」へ移行する処理を追加
- 移行したJANへ空のメモ欄が付くことを確認
- リストを6個まで作成でき、7個目を拒否することを確認
- タブ名変更とリスト削除の単体検査を追加
- スキャンボタンを画面下部中央へ固定
- JANごとのバーコード横へメモ欄を配置
- 読み取り確定または登録済みJANの検出後にカメラを停止
- 新規保存キーは`jan-scan-memo-v2`、旧データは初回保存後に移行

## バーコード表示

- 電動工具・チェーンソー・農薬検索で商品バーコードを直接表示
- JAN文字行と表示ボタンを非表示化
- 画面付近のバーコードだけを生成し、同じJANのSVGを再利用
- バーコードのタップでJANをコピー

## 複数作物検索（1.0.4）

- 2作物・3作物で農薬名の積集合を抽出する単体試験を追加
- 同じ農薬名について作物ごとの複数適用条件を保持
- 実データの「すいか × なす」で園芸グループ共通農薬17品を抽出
- ベニカAスプレーを含み、両作物の条件が保持されることを確認
- 結果は30製品ずつ追加表示

## PWA

- Web App Manifestの向きを`portrait-primary`へ変更
- Service Workerと`version.json`を1.0.5へ更新
- JANリスト保存処理を基本キャッシュへ追加

## 実機確認が必要な項目

- インストールしたPWAで縦向き固定が維持されること
- 6タブの切り替え、名前編集、削除の操作感
- バーコード横のメモ入力領域の幅
- スキャン確定後にカメラ表示が閉じ、先頭のJANへ戻ること
- ハンディ端末による画面上のバーコード読取

この実行環境では実機カメラを使用できないため、カメラ権限、端末ごとの向き固定、読取後の画面遷移は公開後の実機確認が必要です。
